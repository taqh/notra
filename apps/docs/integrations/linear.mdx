Connect your Linear workspace to Notra to use completed issues, projects, and development cycles as data sources for AI-powered content generation.

## What you can do

Once connected, the Linear integration enables Notra to:

- Pull completed issues from your Linear workspace
- Use issue data alongside GitHub activity for richer changelogs and blog posts
- Include project and cycle context in generated content
- Filter by team when your workspace has multiple teams

## Setup

1. Navigate to the **Integrations** page in your Notra dashboard.

2. Click **Connect** on the Linear card, then click **Add Integration** in the dialog.

3. You will be redirected to Linear to authorize read access to your workspace.

4. Once authorized, the integration is created automatically and you are redirected back to Notra.

## Usage

After connecting, Linear issues appear as a data source when creating content:

- In the **Create Content** dialog, select your Linear workspace from the **Integrations** combobox.
- Completed issues within the selected timeframe will appear in the review step with checkboxes.
- In **Schedules**, all enabled Linear integrations are automatically included.

## Limits

- Only one connection per Linear workspace per Notra organization.
- The integration requests read-only access to your workspace.
