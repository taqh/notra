---
title: "Get Single Post"
description: "Get a specific post by ID."
---

`GET /v1/:organizationId/posts/:postId`

Returns a single post for the given organization.
By default, this endpoint only returns published posts.

## Path Params

<ParamField path="organizationId" type="string" required>
  Required organization ID.
</ParamField>

<ParamField path="postId" type="string" required>
  Required post ID.
</ParamField>

## Query Params

<ParamField query="status" type="draft | published" default="published">
  Filter by post status. Repeat this parameter to include both statuses
  (`?status=draft&status=published`).
</ParamField>

## Example Request

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.usenotra.com/v1/YOUR_ORGANIZATION_ID/posts/YOUR_POST_ID?status=draft"
```
