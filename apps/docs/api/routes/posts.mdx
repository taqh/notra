---
title: "Get All Posts"
description: "List posts for an organization."
---

`GET /v1/:organizationId/posts/`

Returns a paginated list of posts for the given organization.
By default, only published posts are returned.

## Path Params

<ParamField path="organizationId" type="string" required>
  Required organization ID.
</ParamField>

## Query Params

<ParamField query="sort" type="asc | desc" default="desc">
  Sort order.
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Number of posts per page. Min `1`, max `100`.
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number to fetch. Min `1`.
</ParamField>

<ParamField query="status" type="draft | published" default="published">
  Filter by post status. Repeat this parameter to include both statuses
  (`?status=draft&status=published`).
</ParamField>

## Example Request

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://api.usenotra.com/v1/YOUR_ORGANIZATION_ID/posts/?sort=desc&limit=10&page=1&status=draft"
```
